# 기능 목록

본 문서는 '요식업 자영업자를 위한 AI 컨설팅 서비스'의 핵심 기능들을 유저 플로우 단계에 따라 정리한 것입니다.

## 1. 온보딩 (Onboarding)

사용자가 서비스에 처음 가입하고 기본 정보를 설정하는 단계입니다.

### 1.1. 사용자 프로필 입력
- **설명**: 서비스 개인화를 위한 사용자의 기본 정보를 수집합니다. **수집된 MBTI, 전직업 정보는 사용자의 성향을 파악하여 더욱 개인화된 콘셉트를 추천하는 데 활용됩니다.**
- **입력 항목**:
  - 나이 (필수)
  - 성별 (필수)
  - MBTI (선택)
  - 전직업 (선택)
  - 창업 경험 여부 (필수)
- **기술**: 입력된 정보는 `PUT /api/v1/users/{userId}/profile` API를 통해 서버에 저장됩니다.

### 1.2. 신규 사용자 튜토리얼
- **설명**: 서비스에 처음 가입한 사용자에게 핵심 기능을 안내하여 서비스 적응을 돕습니다.
- **구성 요소**:
  - 주요 서비스 기능 하이라이트
  - 핵심 화면 스크린샷 안내
  - 첫 프로젝트 생성 과정 가이드
- **동작**: 프로필 입력을 완료한 신규 사용자에게만 표시되며, 완료 후 대시보드로 이동합니다.

## 2. 대시보드 (Dashboard)

사용자가 자신의 프로젝트 현황을 한눈에 파악하고 다음 행동을 결정하는 메인 화면입니다.

### 2.1. 대시보드 UI
- **설명**: 사용자 정보, 프로젝트, 알림 등 핵심 정보를 통합적으로 제공합니다.
- **구성 요소**:
  - 개인화된 환영 메시지
  - 핵심 성과 지표(KPI) 요약 (진행 중 프로젝트 수, **로드맵 진행률** 등)
  - 프로젝트 목록 (카드 그리드 형태)
  - 다음 할일(Next Actions) 섹션
  - 알림 섹션
  - AI 추천 콘텐츠 (창업 관련 아티클, 성공 사례 등)

### 2.2. 프로젝트 관리
- **설명**: 사용자가 생성한 모든 프로젝트를 확인하고 접근할 수 있습니다.
- **기능**:
  - "새 프로젝트 생성" 버튼
  - 기존 프로젝트 카드 클릭 시 해당 프로젝트 상세 페이지로 이동

## 3. 프로젝트 생성 (Project Creation)

사용자가 새로운 창업 또는 컨설팅 프로젝트를 시작하는 기능입니다.

### 3.1. 신규 프로젝트 생성 폼
- **설명**: 모달(Modal) 또는 별도 페이지 형태의 프로젝트 생성 양식을 제공합니다.
- **입력 항목**:
  - 프로젝트명 (필수)
  - 프로젝트 타입 (필수, 라디오 버튼 또는 드롭다운)
    - `NEW_STARTUP`: 예비 창업자
    - `CONSULTING`: 기존 사업 컨설팅
  - 프로젝트 설명 (선택, TextArea)
- **기술**: `POST /api/v1/projects` API를 호출하여 프로젝트를 생성하고, 응답으로 받은 `projectId`를 이용해 다음 단계로 리다이렉트합니다.

## 4. 초기 정보 수집 (Initial Information Collection)

프로젝트에 필요한 구체적인 정보를 단계별로 수집하여 분석의 기반을 마련합니다.

### 4.1. 정보 입력 마법사(Wizard)
- **설명**: UI를 통해 사용자로부터 상세 정보를 입력받습니다. 각 단계 완료 시 정보가 부분 저장됩니다.

### 4.2. 세부 단계별 기능
- **Step 1: 업종 선택**:
  - 14개의 요식업 카테고리 제공
  - 선택한 업종의 상세 정보 표시
- **Step 2: 세부 분류 선택**:
  - 선택한 업종에 맞는 3~5개의 세부 분류 카드 제시
  - 최대 3개까지 다중 선택 가능
- **Step 3: 지역 선택**:
  - 서울시 동 단위 지역 선택 (최대 3개)
  - 선택한 업종에 대해 해당 동의 상권의 유불리 표시
  - 키워드 기반 지역 검색 기능
- **Step 4: 비전 및 가치 설정(옵션)**:
  - 설명: 사용자가 구상하는 가게의 핵심 가치나 비전을 입력받아 AI 추천에 반영합니다.
  - 입력 방식: 키워드 태그 선택 (예: #따뜻한, #가성비, #프리미엄) 또는 자유 텍스트 입력
- **Step 5: 자본금 입력**:
  - 사전 정의된 범위 선택 또는 직접 입력 방식 제공
- **Step 6: 분석 준비 및 시작**:
  - 이전 단계까지 입력한 모든 정보 요약 표시
  - "로드맵 시작" 버튼 클릭 시 다음 단계로

## 5. 콘셉트 추천 및 선택 (Concept Selection & AI Recommendation)

상권분석 결과를 바탕으로 AI가 최적의 사업 콘셉트를 추천합니다.

### 5.1. AI 기반 콘셉트 생성
- **설명**: 업종, 지역, 예산 데이터와 함께 **사용자의 성향(MBTI, 전직업 기반) 및 입력된 비전**을 종합적으로 분석하여 AI 에이전트가 3~5개의 맞춤형 콘셉트를 생성합니다.

### 5.2. 콘셉트 카드 UI
- **설명**: 생성된 각 콘셉트의 상세 정보를 카드 형태로 제공합니다.
- **카드 구성 요소**:
  - 대표 이미지 (DALL-E 생성 또는 스톡 이미지)
  - 콘셉트명 및 상세 설명
  - 핵심 키워드 태그

### 5.3. 콘셉트 상세 보기
- **설명**: 사용자가 콘셉트 카드를 선택했을 때, 사용자의 신뢰도 상승을 위해 추천 위치의 지도 기반 기존 상권 분석 서비스를 iframe으로 가져와 보여줍니다. 또한 상권 데이터와 콘셉트를 기반으로 AI 에이전트가 상세 분석을 표시합니다.

## 6. 로드맵 자동 생성 (Roadmap Auto-Generation)

선택된 콘셉트를 기반으로 창업 준비부터 실행까지의 전체 과정에 대한 로드맵을 자동으로 생성합니다.

---

## 7. 로드맵 상세 및 실행 관리 (Detailed Roadmap & Execution Management)

선택된 콘셉트와 사용자의 정보를 바탕으로 생성된 창업 로드맵의 세부 기능입니다. 각 단계는 창업 준비에 필요한 핵심 과업들을 주제별로 그룹화하여 안내하며, **실행 관리를 위한 공통 기능을 포함합니다.**

### 7.1. 실행 관리 공통 기능
- **설명**: 로드맵의 모든 단계에 공통으로 적용되어 사용자가 창업 과정을 체계적으로 관리하고 실행할 수 있도록 돕습니다.

### 7.2. 공간 기획 (Space Planning)
- **설명**: 선택한 사업 콘셉트에 맞춰 인테리어와 간판 디자인의 방향성을 시각 자료와 함께 제안하고, 관련 절차를 안내합니다.
- **주요 기능**:
  - 인테리어 콘셉트 제안 (AI 이미지 포함)
  - 간판 디자인 제안 (AI 이미지 포함)
  - 프로세스 가이드

### 7.3. 운영 준비 (Operations Preparation)
- **설명**: 안정적인 매장 운영을 위한 식자재 공급처 확보와 브랜드 정체성을 담은 패키지 제작을 지원합니다.
- **주요 기능**:
  - 식자재 공급처 추천
  - 패키지 디자인 및 제작 가이드 (AI 이미지 포함)
  - 비용 절감 방안 제시

### 7.4. 자금 계획 (Financial Planning)
- **설명**: 사용자의 자본금과 목표 금액에 맞춰 정부 정책자금 정보를 제공하고, 자금 확보 계획 수립을 돕습니다.
- **주요 기능**:
  - 맞춤형 정책자금 추천
  - 자금 확보 시뮬레이션
  - 사업계획서 작성 지원

### 7.5. 입지 선정 (Location Selection)
- **설명**: 초기 정보 수집 단계의 데이터를 활용하여 최적의 입지 조건을 제시하고, 계약 과정의 주의사항을 안내합니다.
- **주요 기능**:
  - 맞춤형 입지 조건 제시
  - 계약 전 확인사항 체크리스트
  - 계약 프로세스 안내

### 7.6. 행정 및 인허가 (Administration & Permits)
- **설명**: 창업에 필요한 각종 인허가 및 행정 절차를 누락 없이 진행할 수 있도록 개인화된 정보를 제공합니다.
- **주요 기능**:
  - 필수 절차 목록화
  - 준비물 및 진행 방법 안내
  - 관련 기관 정보 제공

### 7.7. 메뉴 개발 (Menu Development)
- **설명**: 성공적인 메뉴 구성을 돕고, 수익성 분석을 위한 원가 계산 기능을 제공합니다.
- **주요 기능**:
  - 시그니처 메뉴 추천 (AI 이미지 포함)
  - 레시피 및 원가 관리
  - 메뉴 확정 가이드
